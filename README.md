## Getting Started

### Prerequisites

- Node.js (if applicable)
- npm or yarn

### Installation

Clone the repository:

```sh
git clone https://github.com/Nong-Duy-Tran/Grocery-Store.git
cd grocery-store
```

Install dependencies:

```sh
npm install
```

### Running the Application

```sh
npm start
```
